마크다운 기본 구문 / Markdown Basic Syntax

Feb 15, 2019

Markdown문법은 어렵지 않다. HTML에 비해 아주 짧고 간단명료하다. Jekyll과 GitHub로 블로그를 운영하며 자연스럽게 Markdown과 마주하게되는데.. HTML의 기본지식과 이해만 있다면 정말 너무 쉬운 언어다.

Markdown파일에 HTML 코드를 붙여넣어도 전부 적용이 된다. 그치만 나는 전문 프로그래머도 아닌 일반인이고.. 그저 배움의 뜻에서 Jekyll 운용에 있어서 적극 Markdown코드를 사용하려고 한다.

사용빈도가 낮을수록 쓰려고할때, 생각나지 않을때 구글링하지 않고 바로바로 찾아 꺼내쓰기위해서 첫포스팅을 마크다운 구문. ‘Markdown Syntax’로 하기로했다.

*Markdown
*HTML
*Jekyll
*GitHub


1. Headings

Markdown

# H1
## H2
### H3
#### H4
##### h5
###### H6

HTML

<h1>H1</h1>
<h2>H2</h2>
<h3>H3</h3>
<h4>H4</h4>
<h5>H5</h5>
<h6>H6</h6>

Markdown

안녕하세요

Markdown

4. Bold

Markdown

**볼드** 안볼드
__볼드__ 안볼드

HTML

<p><strong>볼드</strong> 안볼드</p>
<p><strong>볼드</strong> 안볼드</p>

결과

볼드 안볼드


5. Italic

Markdown

_이탤릭_ 안이탤릭
*이탤릭* 안이탤릭

HTML

<p><em>이탤릭</em> 안이탤릭</p>

결과

이탤릭 안이탤릭

Markdown

7. List

Markdown

1. 하나
2. 둘
3. 셋
    1. 셋 하나
    2. 셋 둘
4. 넷

HTML

    <li>하나</li>
    <li>둘</li>
    <li>셋</li>
        <ol>
            <li>셋 하나</li>
            <li>셋 둘</li>
        </ol>
    <li>넷</li>
</ol>
