Jekyll Install Mac OS X / 지킬 설치하는 법 for 맥

준비물

  • Ruby (v2.2.5 이상)
  • RubyGems
  • root dir 설정할 경로


1. 터미널에서 ruby 버젼 확인

$ ruby -v

으로 ruby 버젼 확인 2.2.5 이상

2. 터미널에서 RubyGems 버젼 확인

$ gem -v

으로 gem 설치 확인

3. jekyll 설치할 경로 생성하기

ex) /Users/fillip/desktop/jekyll

데스크탑 위치에 ‘jekyll’ 이름의 폴더를 생성후 이곳에 jekyll을 설치할 예정

4. Jekyll 설치

$ gem install jekyll bundler
$ jekyll new myblog
$ cd myblog
$ bundle exec jekyll serve
  • 첫번째 커맨드라인을 실행시키면 ruby의 gems폴더 안에 jekyll bundle이 설치됨.
  • 두번째 커맨드라인을 실행시키면 /Users/fillip/desktop/jekyll 폴더 안에 ‘myblog’라는 이름의 폴더가 생성됨. (생성하고싶은 이름으로 생성하면됨.)
  • 세번째 커맨드라인을 실행시키면 두번째 커맨더라인으로 생성된 ‘myblog’폴더로 이동.
  • 네번째 커맨드라인을 실행시키면 ‘_config.yml’을 포함한, 작업할 blog에 필요한 jekyll 문서들이 생성됨.

  • 이후 ‘_config.yml’ 파일로 blog의 전체적인 세팅, 플러그인을 설정하고, ‘_posts’ 폴더 안에 markdown 문서로 포스팅내용을 작성을 후

  • /Users/fillip/desktop/jekyll/myblog 위치에서 $ jekyll serve라는 터미널명령어로 jekyll을 실행시켜주면,

  • ‘_site’폴더 안에 jekyll에 의해 자동으로 생성된 html문서들을 볼 수 있다.

디테일한 CSS나 플러그인 설정은 ~ruby/gems/minima에서 관리.

_config.yml문서를 열어 ‘theme’의 이름을 확인 (ex: minima) 터미널에서 커맨드명령어로 ‘bundle show minima’

예로 /Users/fillip/.rvm/gems/ruby-2.6.1/gems/minima-2.5.0 라는 경로가 나오면

finder에서 shift+command+g를 동시에 눌러 경로변경에 /Users/fillip/.rvm/gems/ruby-2.6.1/gems/minima-2.5.0 를 붙여넣고 경로이동.

includes, _layouts, _sass, assets 폴더가 이곳에 존재한다.
